Modules in js can be exported and imported by Common Js method or by ES module method 
In common Js, each file is considered as a module
To export a function you must use module.exports 

ES Module: 
Supported in node version 14 and above 
The exports an be named or default 
We use exports keyword instead of module.exports\


There are three types of modules: build-in, local, third-party modules 
1. Build-in modules: fs, http, events, path, streams
